# Тест-кейсы для REST API GET: /products/{productId}/status

## Общая информация
Метод: GET  
Авторизация: `authToken` (16-символов строка, состоит из букв и цифр)  
Параметры: `recalculate`, `owner`, `region` (необязательные)

---

## Позитивные кейсы

| ID | Название | Описание | Ожидаемый результат |
|----|----------|----------|---------------------|
| TC001 | Валидный запрос (все параметры) | Все параметры указаны корректно | 200 + JSON { "productStatus": 1 } |
