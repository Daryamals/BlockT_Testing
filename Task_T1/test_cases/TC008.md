# TC008: Обработка внутренней ошибки сервера

**Предварительные шаги**:
1. Создать тестовый продукт с ID 1234 или убедиться, что он существует в системе.

**Шаги**:
1. Отправить GET-запрос на `/products/1234/status` с валидными параметрами:
   - `authToken=testEqual16simvl` - пользователь не имеет права доступа к нему
   - `recalculate=true`
   - `owner=Создатель`
   - `region=Северо-Запад`

**Ожидаемый результат**:
- HTTP: 403
- Тело ответа: `{ "errorMessage": "Доступ запрещен" }`.

**Серьезность:** 
High
