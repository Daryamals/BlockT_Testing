# TC005: Обработка несуществующего productId

**Предварительные шаги**:
1. Создать тестовый продукт с ID 8888 или убедиться, что не существует в системе.

**Шаги**:
1. Отправить GET-запрос на `/products/8888/status` с параметрами:
   - `authToken=testEqual16simvl`
   - `recalculate=true`
   - `owner=Создатель`
   - `region=Северо-Запад`

**Ожидаемый результат**:
- HTTP 404
- Тело ответа: {"errorMessage": "Product not found"}

**Severity:** 
Medium
