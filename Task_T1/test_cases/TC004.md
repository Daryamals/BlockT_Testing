# TC004: Невалидный запрос при отсутствии authToken

**Предварительные шаги**:
1. Создать тестовый продукт с ID 1234 или убедиться, что он существует в системе.

**Шаги**:
1. Отправить GET-запрос на `/products/1234/status` с параметрами:
   - `recalculate=true`
   - `owner=Создатель`
   - `region=Северо-Запад`

**Ожидаемый результат**:
- HTTP: 401
- Тело ответа: `{"errorMessage": "Нет ввода обязательного параметра authToken для авторизации"}`

**Severity:** 
High
