# TC010: Обработка запроса с истёкшим, но валидным по формату токеном авторизации

**Предварительные шаги**:
1. Создать тестовый продукт с ID 1234 или убедиться, что он существует в системе.

**Шаги**:
1. Отправить GET-запрос на `/products/1234/status` с валидными параметрами:
   - `authToken=testEqual16simvl` - истекший токен
   - `recalculate=true`
   - `owner=Создатель`
   - `region=Северо-Запад`
   

**Ожидаемый результат**:
- HTTP 403
- Тело ответа: {"errorMessage": "Истёк срок действия токена"}

**Серьезность:**
Medium (успешный сценарий, влияет на основную функциональность).
