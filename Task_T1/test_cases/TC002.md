# TC002: Успешное получение статуса неготового продукта с валидным authToken и всеми параметрами

**Предварительные шаги**:
1. Создать тестовый продукт с ID 1234 или убедиться, что он существует в системе.

**Шаги**:
1. Отправить GET-запрос на `/products/1234/status` с одним параметром:
   - `authToken=testEqual16simv2`
   - `recalculate=false`
   - `owner=Пользователь`
   - `region=Сибирь`
     
**Ожидаемый результат**:
- HTTP 200
- Тело ответа: `{"productStatus": 0}`

**Серьезность:**
High.
