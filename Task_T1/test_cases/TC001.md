# TC001: Успешное получение статуса готового продукта с валидным authToken и всеми параметрами

**Предварительные шаги**:
1. Создать тестовый продукт с ID 1234 или убедиться, что он существует в системе.

**Шаги**:
1. Отправить GET-запрос на `/products/1234/status` с валидными параметрами:
   - `authToken=testEqual16simvl`
   - `recalculate=true`
   - `owner=Создатель`
   - `region=Северо-Запад`
   

**Ожидаемый результат**:
- HTTP 200
- Тело ответа: {"productStatus": 1}

**Серьезность:**
Medium (успешный сценарий, влияет на основную функциональность).
