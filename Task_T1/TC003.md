# TC003: Невалидный authToken

**Предварительные шаги**:
1. Создать тестовый продукт с ID 1234 или убедиться, что он существует в системе.

**Шаги**:
1. Отправить GET-запрос на `/products/1234/status` с параметром:
   - `authToken=(см "Ожидаемый результат")`

**Ожидаемый результат**:
- HTTP 500
  
|  № | Вводимое значение | Ожидаемый результат |
|----|------------------|---------------------|
| 1 | Less16simvol | `{"errorMessage": "Ошибка: длина меньше 16 символов"}` |
| 2 | testMore16simvols | `{"errorMessage": "Ошибка: длина больше 17 символов"}` |
| 3 | testNoHaveSimvol | `{"errorMessage": "Ошибка: не включает в себя цифры"}` |
| 4 | 1234567891012345 | `{"errorMessage": "Ошибка: не включает в себя буквы"}` |
| 5 | HaveNot1NotLet__ | `{"errorMessage": "Ошибка: содержит символы кроме букв и цифр"}` |
