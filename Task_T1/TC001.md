# TC001: Валидный запрос (все параметры)

**Предварительные шаги**:
1. Создать тестовый продукт с ID 1234 или убедиться, что он существует в системе.

**Шаги**:
1. Отправить GET-запрос на `/products/1234/status` с параметрами:
   - `authToken=testEqual16simvl`
   - и комбинацию параметров ниже:
   
|  № | recalculate | owner | region | Тело ответа |
|----|----------|----------|---------|--------------|
| 1 | true | Создатель| Северо-Запад | {"productStatus": 1} |
| 2 | false | Пользователь| Сибирь | {"productStatus": 0} |
| 3 | null | null| Поволжье | {"productStatus": 0} |
| ... | ... | ... | ... | ... |

**Ожидаемый результат**:
- HTTP 200
- Тело ответа: {"productStatus": 1} или {"productStatus": 0} в зависимости от бизнес-логики.
